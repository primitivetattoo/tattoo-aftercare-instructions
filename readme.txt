Interactive tattoo aftercare timeline with personalized day tracker, printable instructions, and studio contact info.

== Description ==

A beautiful, interactive aftercare guide that tattoo studios can embed on their website. Clients select their tattoo date and get personalized healing phase tracking with step-by-step instructions.

Features:

* 5-phase aftercare timeline (Day 0 through Long-term)
* Personalized day tracker — "When did you get your tattoo?"
* Remembers the date between visits (localStorage)
* Highlights current healing phase with progress indicators
* Completed phases show green checkmarks
* Printable version — opens all phases for clean print
* Emergency warning section for signs of infection
* Studio contact card with phone and email
* Fully customizable phases, instructions, and colors
* Dark-themed, mobile-responsive design
* Simple shortcode: [tattoo_aftercare]

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate through the Plugins menu
3. Configure at Settings → Tattoo Aftercare
4. Add `[tattoo_aftercare]` shortcode to any page
